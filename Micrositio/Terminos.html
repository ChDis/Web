            HTML: es el acronimo para "Hyper Text Markup Language" que es el lenguaje estandar para la
            creacion de paginas web, este describe la estructura de una pagina web.

            Elemento: es cualquier texto o contenido entre dos etiquetas.

            HTML5: es la ultima version de HTML y la que se usa actualmente.

            Atributos: proveen informacion adicional acerca de los elementos. 

            CSS: es usado para darle forma a la pagina web, se pueden controlar fuentes, colores, espaciado 
            entre elementos etc.
