Automatizadores

Las organizaciones cada vez tienen mas necesidad de saber los procesos internos para mejorarlos ahorrando costes y tiempo. Esto no solo ahorra costes y tiempo sino que tambien se encarga de mejorar diversos aspectos como la agilidad operacional o la productividad.
Este cumple en el paradigma de DevOps debido a las formas en como puede mejorar la productividad de un sistema y tambien gracias a que ayuda a desarrollar softwares mas rapidos, con mayor calidad y con un coste menor a que si se tuviera que programar todo manualmente.
Tambien tiene un papel importante en los sistemas con CI/CD, puesto que este se basa en la distribucion de aplicaciones con el uso de automatizacion en las etapas de desarrollo.
